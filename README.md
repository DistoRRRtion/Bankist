# Bankist is a online banking app 

![bankist app](https://user-images.githubusercontent.com/99020542/189517183-a19ace79-e977-4679-a0ef-7782119e5573.gif)

## Its main features:
    - Log-in
    - Transfer To Other Accounts
    - Request Loan
    - Delete Account
    - Log-out Timer
    - Sort Movements
    - Internationalize Date & Currency
